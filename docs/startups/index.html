Investments#

Eventually I want to run my own VC fund, so I’ve been practicing through Reg-CF. Hopefully one day this will serve as a track record when I go to raise money from investors, but in the meantime it’s teaching me a lot and reading the quarterly updates is always exciting. I initially set out to make 100 investments by the time I graduate college, so far I’m 3 years in with 51 investments across 49 companies. Been slowing down lately to focus on building out Aeternum, so 100 seems unlikely, but I’m satisfied with my progress so far and learned a lot.
